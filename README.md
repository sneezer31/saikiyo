# さいきよ (Saikiyo)

## 概要
本リポジトリは、ブラウザ上で稼働する専用の短歌・リスト編纂システムです。
モバイル端末からの操作に最適化され、思考を即座に記録し、編むための機能を提供します。

## 主な機能
*   **完全ローカル駆動:** 全てのデータはブラウザの `localStorage` に保存されます。
*   **3列マトリクスUI:** 入力コンソール、タイムライン、各種設定・編集パネルの独立したレイアウト。
*   **リスト制作机:** 選択したコンテント群を任意に組み合わせ、タイトルとメモを付与してリスト化する機能。
*   **鑑賞モード:** 作成したリストやテキストを、視界を塞ぐUIなしで読解するための専用画面。

## 備考
本システムは私的利用を目的として極限までチューニングされています。
